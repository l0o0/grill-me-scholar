# Grill Me Scholar

一个自适应学术访谈 skill，帮助研究者把模糊兴趣逐步收敛为一个聚焦、可行且证据边界清晰的研究主题。

许多研究项目开始时，研究者还没有精确的问题。宽泛兴趣并不等于研究主题，一个听起来新颖的题目也不代表它已经成立。Grill Me Scholar 通过一次只问一个问题的访谈逐步收窄研究空间，同时把关键取舍留给研究者本人。

## 为什么需要它

普通的选题生成提示经常过早给出一组包装完整的题目。它们可能擅自选择研究人群或方法，忽略数据访问和伦理限制，或者把未经核查的说法重复成“研究空白”。

Grill Me Scholar 会放慢这个过程。它让不确定性保持可见，并在确认方向之前建立双方对研究问题的共同理解。

## 核心特点

- **一次处理一个决策：** 活跃访谈回合只提出一个非复合问题。
- **自适应入口：** 针对完全没方向、只有宽泛领域或已有候选题目采用不同起点。
- **研究者拥有最终决定：** 提供判断框架与建议，但不擅自决定范围、方法或价值取向。
- **证据纪律：** 将主张标记为 `verified`、`user-provided` 或 `needs verification`。
- **学术可行性检查：** 检查概念、证据、方法、资源、访问条件、伦理、贡献与反论。
- **跨学科镜头：** 针对人文、社科、理工、健康以及设计与实践研究调整检查重点。
- **明确的完成门槛：** 只有当主题连贯、可行、边界清晰并由研究者确认后，才输出最终研究主题卡片。

## 安装

### 使用 `npx skills` 安装

```bash
npx skills add l0o0/grill-me-scholar --skill=grill-me-scholar
```

### 手动安装到项目

```bash
git clone git@github.com:l0o0/grill-me-scholar.git
mkdir -p /path/to/project/.agents/skills
cp -R grill-me-scholar/.agents/skills/grill-me-scholar /path/to/project/.agents/skills/
```

### 手动安装为 Codex 个人 skill

```bash
mkdir -p ~/.agents/skills
cp -R grill-me-scholar/.agents/skills/grill-me-scholar ~/.agents/skills/
```

不同 Agent Skills 兼容工具的 skill 发现路径可能不同。如果你的客户端使用其他路径，请以该客户端的约定为准。

## 快速开始

English:

```text
Use $grill-me-scholar to help me narrow and validate my research topic.
```

中文：

```text
使用 $grill-me-scholar 帮我收窄并验证研究主题。
```

你可以在完全没有题目、只有宽泛领域或已有候选问题时开始。也可以要求 skill 暂停、回退、解释为什么要问某个问题，或者输出临时版本。

## 访谈示例

> **研究者：** 我想研究社交媒体如何影响大学生。
>
> **Grill Me Scholar：** “影响”可能指向多种结果。你最关心哪一个结果？
>
> **研究者：** 学生焦虑。
>
> **Grill Me Scholar：** 焦虑现在是暂定结果，但研究目标仍不明确。你想考察变量之间的关联，还是理解学生的主观体验？
>
> **研究者：** 我觉得没人研究过短视频和焦虑，所以这就是我的研究空白。
>
> **Grill Me Scholar：** 这是一个 `user-provided` 主张，目前仍然 `needs verification`，还不能视为成立的研究空白。你目前有哪些文献检索证据支持它？

访谈会持续到研究对象、范围、证据、方法、约束、伦理和贡献足够清晰。它不会强制每次会话回答固定数量的问题。

## 研究主题卡片

研究者最终确认后，skill 会输出包含以下内容的结构化卡片：

```text
暂定题目
一句话研究主题
核心研究问题与必要的子问题
研究对象、范围与关键概念
研究动机与潜在贡献
与已有研究的暂定关系
建议的方法路径
所需材料、数据与条件
可行性与伦理风险
已核查证据与来源
待验证假设
下一步最小行动
备选方向及未采用原因
```

如果主题暂时无法收敛，skill 会报告当前最佳候选、阻塞性不确定因素和最小验证任务，而不会制造虚假的确定性。

## 学科镜头

核心访谈流程保持一致，但检查重点会根据学科调整：

- **人文学科：** 材料、语境、解释框架与文本边界。
- **社会科学：** 概念、机制、案例、变量、数据访问、推断与参与者伦理。
- **理工学科：** 系统、假设、测量、控制、不确定性、安全与可复现性。
- **医学与健康：** 人群、暴露或干预、结局、偏倚、隐私、访问条件与伦理审查。
- **设计与实践研究：** 场景、利益相关者、产物或干预、参与过程与评价标准。
- **跨学科研究：** 各学科在理论、方法、证据或语境中的作用，以及彼此可能冲突的证据假设。

## 工作原理

skill 会跟踪兴趣、现象、对象、范围、证据、方法、资源、伦理、贡献与投入意愿。每个维度可能处于未知、暂定、已确认、待验证或相互冲突的状态。

下一个问题会针对最可能改变或否定研究主题的不确定因素。如果后续约束推翻了先前方案，例如数据无法访问或伦理审查周期过长，访谈会回退并重新设计路径，同时尽可能保留最初的研究动机。

只有当代理已经检查可靠来源、确认它直接支持具体主张并记录可追溯位置时，该主张才属于 `verified`。仅仅存在一个可以解析的 DOI 并不足够。

## 适用范围与限制

Grill Me Scholar 用于形成和压力测试研究主题。它不会：

- 撰写完整研究计划；
- 保证题目或研究空白具有新颖性；
- 替代系统性文献综述；
- 替代导师、领域专家、统计学家或方法专家；
- 授予数据访问、参与者同意或伦理审批；
- 替研究者决定兴趣、价值取向或资源投入。

输出的可靠性取决于访谈过程中可获得的信息和证据。重要事实仍应根据适当的学术来源和机构要求进行核查。
